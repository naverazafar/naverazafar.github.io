Projects with Companies

TeraDrive: I worked with six individuals on a project for TeraDrive. TeraDrive is a professional data recovery company based in BC. TeraDrive wanted to have a tool that would help them distinguish which files can be recovered and which of their files are too corrupted to be recovered. We created a tool that scanned the files and folders chosen and outputted an HTML file that showed which of those files were corrupted, not corrupted, and unscanned. We created the program using Python, JavaScript, HTML, and CSS. We used the template for the Snap2HTML application as our base code. We packaged the application using py2exe.
